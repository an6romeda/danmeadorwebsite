# danmeador.work

Personal site. Privacy lawyer. This is not a resume.

## What's here

A single-page site with four sections:

- **About** — The short version of a longer story
- **Side quests** — Things I'm building
- **Library** — Books, podcasts, articles shaping my thinking
- **Contact** — Where to find me

## Built with

HTML, CSS, JavaScript. No frameworks, no build step. Just files.

## Run locally

```bash
python -m http.server 8000
# or
npx serve
```

## Details

- Typewriter effect sets the tone
- Project cards open modals with live site previews
- Library toggles between current and past influences
- Navigation highlights based on scroll position
- Responsive down to mobile
